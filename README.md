Agency Jekyll theme
====================

Agency theme based on [Agency bootstrap theme ](https://startbootstrap.com/template-overviews/agency/)

# 如何使用 How to use

###文件夾 Portfolio 

作品集項目位於“/_posts”目錄下。 Portfolio projects are in '/_posts'

圖片位於“/img/portfolio”目錄下。 Images are in '/img/portfolio'

###關於 About

圖片位於“/img/about/”目錄下。 Images are in '/img/about/'

###團隊 Team

團隊成員和資訊位於「_config.yml」檔案中。 Team members and info are in '_config.yml'

圖片位於“/img/team/”目錄下。 Images are in '/img/team/'
